# Contributing

Thanks for your interest in improving Codex CLI Launcher.

## Development

```bash
npm install
npm run check
```

Keep changes focused and covered by tests. Do not add official OpenAI or Codex logos, marks, screenshots, or branding assets unless you have permission to use them.

## Pull Requests

- Keep user-facing behavior documented in `README.md`.
- Add or update tests for launcher behavior, package metadata, and install helpers.
- Run `npm run check` before submitting changes.
